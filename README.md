# SDDS R$

```
git clone git@github.com:vstabile/sdds-rs.git
git cd sdds-rs
```

## Instalação local

Pré-requisito: Node.js

```
npm install
npm start
```

## Instalação Docker

Pré-requisito: Docker

```
docker build -t sdds/sdds .
docker run -p 3000:3000 -d sdds/sdds
```
